## Completed: Auto-Zoom Effects Stage with Face-Aware Zooming

### Created Files:
1. **server/src/pipeline/stages/effects.ts**
   - Implemented effectsStage implementing PipelineStage interface
   - Auto-zoom effect triggered by audio peak moments (type: 'peak' or 'sustained')
   - Face-aware zooming when face tracking data is available
   - Configurable zoom intensity: 'subtle' (1.2x), 'medium' (1.3x), 'strong' (1.5x)
   - Built FFmpeg zoompan filter with dynamic zoom and pan expressions
   - Smooth zoom transitions: zoom in (0.5s) → hold (1.0s) → zoom out (0.5s)
   - Filters audio moments to only process moments within each clip's time range
   - Falls back to center zoom when no face tracking data available

### Features Implemented:
- Auto-zoom on audio peaks with configurable intensity
- Face-aware zooming that pans toward detected faces during zoom
- Smooth zoom transitions using FFmpeg's zoompan filter
- Linear interpolation for zoom factor and face position
- Clip-relative timestamp conversion for accurate moment timing
- Graceful fallback when auto-zoom is disabled or no moments detected
- Integration with existing audio analysis pipeline
- Support for face tracking data from reframe stage

### Technical Details:
- Uses FFmpeg's zoompan filter for hardware-accelerated zoom effects
- Builds complex filter expressions with nested if statements for interpolation
- Normalizes face positions to 0.0-1.0 range for resolution-independent panning
- Filters moments by clip start time and duration for accurate sync
- Generates 4 keyframes per zoom moment (start, peak, hold-end, end)
- Sorts and manages keyframes chronologically
- Interpolates face positions between keyframes using linear interpolation
- Preserves clip metadata (originalPath, clipId) through the pipeline
- Updates progress to 70% (positioned between reframe at 60% and caption at 80%)

### Zoom Behavior:
- Triggers on audio moments with type 'peak' or 'sustained'
- Zoom sequence: 1.0x → maxZoom over 0.5s → hold 1.0s → 1.0x over 0.5s
- Center pan formula: x = iw/2-(iw/zoom/2), y = ih/2-(ih/zoom/2)
- Face-aware pan: x = iw/2-(iw/zoom/2)+(faceX-0.5)*iw/zoom
- Face position interpolated between tracking keyframes for smooth motion

### Integration:
- Seamlessly integrates into existing pipeline between reframe and caption stages
- Respects job settings: autoZoom boolean and zoomIntensity string
- Passes effectsClips to caption stage for final processing
- Creates /effects/ directory parallel to /reframed/ directory
- Handles clips without audio moments by copying unchanged
