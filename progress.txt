### Implementation Details:
- The `createSplitScreen()` function was already fully implemented in reframe.ts but was never called
- Split-screen uses FFmpeg vstack filter to stack facecam (35% of height) on top of gameplay (65%)
- When splitScreen setting is true, the pipeline checks for dual video sources in context.metadata
- If dual sources (facecamPath and gameplayPath) are available, creates split-screen layout
- If dual sources are not available, falls back to standard center crop reframe
- Maintains target aspect ratio (default 9:16 for vertical videos)
