### Technical Implementation Details:

**Face Detection Flow:**
1. Extract frames at 2 FPS from input video using FFmpeg
2. Initialize MediaPipe FaceDetector with BlazeFace short-range model
3. Process each frame to detect faces
4. For detected faces:
   - Get bounding box coordinates
   - Calculate face center position
   - Generate crop keyframe to center face in output
5. Apply smoothing to keyframes for fluid camera motion
6. Build FFmpeg crop filter with interpolation
7. Clean up temporary files

**Fallback Strategy:**
- If MediaPipe initialization fails: falls back to center crop
- If no faces detected in any frame: falls back to center crop
- If frame processing fails: continues to next frame
- Graceful error handling throughout

**Performance Considerations:**
- Runs at 2 FPS (analyzes 1 frame every 0.5 seconds)
- Minimal overhead for short clips (typical 10-60 second clips)
- Temporary frames cleaned up after processing
- Model loaded from CDN (no local storage required)
