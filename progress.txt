## Task: Add filler word removal

### Changes Made:
- Implemented filler word removal functionality in the transcription module
- Added `removeFillers` and `customFillerWords` configuration options to `TranscribeConfig` interface
- Created `removeFillerWords()` function that processes transcription results to remove common filler words
- Integrated filler removal into both `transcribeWithWhisperAPI()` and `transcribeWithLocalWhisper()` functions
- Implemented smart removal logic that:
  - Removes default filler words: um, uh, umm, uhh, like, you know, i mean, sort of, kind of, basically, actually, literally
  - Preserves filler words after sentence-ending punctuation (., !, ?)
  - Preserves filler words with high confidence (>= 0.95) indicating emphasis
  - Adjusts timestamps to close gaps < 1 second
  - Preserves natural pauses > 1 second
  - Supports custom filler word lists
- Created comprehensive test suite with 14 test cases covering all edge cases

- server/src/captions/transcribe.ts (added removeFillerWords function and updated TranscribeConfig)
